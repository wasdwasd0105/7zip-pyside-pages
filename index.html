7-Zip Archiver is an open-source GUI for 7-Zip, built using Python QT.

Currently designed specifically for MacOS, it's optimized for M1 chips, ensuring smooth and fast performance.

Download

 Mac APP Store Or GitHub Project

Preview Window

With the preview window feature, users can easily view archive contents without the need for extraction, offering a streamlined experience.

Workspace

Use Workspace feature, users can easily display contents of the filesystem and open an archive from the Workspace

Easy Compress

Users can choose compression level and enable password protection

Optimized for Mac M1 Chip

This application is specially optimized for the Mac M1 chip, ensuring enhanced performance and efficiency on the latest Mac hardware.
